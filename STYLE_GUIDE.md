# Design System - Duolingo Style (Dark/Gold Theme)

## Paleta de Colores

- **Color primario vibrante:** Dorado (#FDB42D) para CTAs, acentos y elementos interactivos
- **Fondo principal:** Azul oscuro profundo (#1B2134)
- **Fondos secundarios:** Variaciones ligeramente más claras (#232B42, #2A3550) para tarjetas y secciones alternadas
- **Texto primario:** Blanco (#FFFFFF) o gris muy claro (#E8EAED)
- **Texto secundario:** Gris medio-claro (#B8BCC8) para descripciones
- **Acentos secundarios:** Naranjas cálidos (#FF8C42), verdes esmeralda (#4ECDC4) para estados de éxito

## Tipografía

- Sans-serif moderna, redondeada y amigable (tipo Nunito, Rounded, DM Sans, o Inter)
- Pesos variables: Bold/ExtraBold para títulos, Medium para subtítulos, Regular para cuerpo
- Tamaños jerárquicos claros: Títulos grandes (36-48px), subtítulos (20-24px), cuerpo (16-18px)
- Alto contraste: texto blanco sobre fondo oscuro para máxima legibilidad

## Componentes y Elementos

- **Botones primarios:** fondo dorado (#FDB42D), texto oscuro (#1B2134), bordes redondeados generosos (12-16px), font-weight bold
- **Botones secundarios:** borde dorado, texto dorado, fondo transparente u oscuro
- **Tarjetas:** fondo #232B42, border-radius 16px, sutil borde o glow dorado opcional
- **Sombras:** glow effect sutil con tono dorado para elementos importantes (box-shadow con #FDB42D a baja opacidad)
- **Iconografía:** líneas redondeadas, estilo outline o filled, color dorado o blanco

## Estilo Visual

- Dark mode premium con toques lúdicos
- Flat design 2.0 con profundidad mediante gradientes oscuros sutiles y glows
- Ilustraciones con paleta limitada: dorado, blanco, y acentos de color
- Elementos gráficos lúdicos: badges dorados, barras de progreso con relleno dorado, estrellas brillantes
- Gradientes sutiles en fondos: del azul oscuro a tonos ligeramente más claros o púrpuras profundos

## Layout y Estructura

- Mobile-first y completamente responsivo
- Secciones con fondos alternados (#1B2134 / #232B42)
- CTAs dorados prominentes con alto contraste
- Hero section: título grande en blanco, subtítulo en gris claro, ilustración con toques dorados, botón dorado prominente
- Navegación minimalista: texto blanco/gris claro, hover con tono dorado
- Espaciado generoso (padding amplios) para respiración visual

## Interactividad

- **Hover states:**
  - Botones dorados → ligero brightening + scale(1.05) + sombra dorada más intensa
  - Enlaces → color dorado con underline animado
  - Tarjetas → borde o glow dorado sutil
- Transiciones suaves (0.2-0.3s ease)
- Estados activos con glow dorado
- Loading states con spinner o animación dorada

## Efectos Especiales

- Glow effect dorado para elementos destacados (text-shadow o box-shadow con #FDB42D)
- Gradientes radiales sutiles desde esquinas (dark blue → darker blue)
- Partículas o destellos dorados opcionales para elementos gamificados
- Borders con gradiente dorado para secciones premium

## Tono y Personalidad

- Premium pero accesible
- Sofisticado y lúdico a la vez
- Motivador y energético
- Moderno con toque de lujo

## Accesibilidad

- Contraste WCAG AAA entre texto blanco y fondo oscuro
- Contraste WCAG AAA entre texto oscuro y botones dorados
- Focus states visibles con outline dorado
- Tamaños táctiles mínimo 44x44px para móviles
